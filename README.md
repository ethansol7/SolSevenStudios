# Sol Seven Studios

Premium studio website for Sol Seven Studios.

Built with Vite, React, Framer Motion, React Three Fiber, Three.js, and Web Audio API. The site uses real Sol Seven product and process assets organized under `public/assets`.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

GitHub Pages deployment is configured through `.github/workflows/deploy.yml`.
